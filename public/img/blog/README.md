# Blog Images Directory

This folder contains images for blog posts. 

## Image Guidelines:
- **Aspect Ratios**: 
  - 19:6 ratio (e.g., 1900x600px, 950x300px) - Wide banner style
  - 16:9 ratio (e.g., 1920x1080px, 1280x720px, 800x450px) - Standard widescreen
- **Format**: JPG or PNG
- **Size**: < 1MB for optimal loading
- **Naming**: Use descriptive kebab-case names
